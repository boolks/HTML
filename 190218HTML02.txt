# : id 속성을 지정한 값
. : class 속성을 지정한 값
id : 유일한 값을 가지고 있어야 함
class : 같은 값을 가지고 있어도 상관없음

text-align : 글자 위치 정렬
width, height : 가로 세로 길이 지정
margin, padding : 범위 지정
color : 글자 색 지정
background : 배경색 지정

[id ^= tag] : tag로 시작하는 id 문자열에 대해 스타일 적용
[id $= tag] : tag로 끝나는 id 문자열에 대해 스타일 적용
[id *= tag] : tag를 포함하는 id 문자열에 대해 스타일 적용
