HTML 1일차 강의
---------------------------------------------------------------------------------------------------------------------------------------

HEAD
설정 영역

BODY
편집 영역

-------------------------------------------------------------------------------------------------------------------------------------

Tag

h1~h6 : 헤더라인 굵은 글씨로 표시됨
br : Enter(줄바꿈)
hr : 수평 줄 태그
p : 문단 단락 변경
<a href=""> : 페이지 변환 앵커 태그
b : bold체
i : italic체
del : 취소선
sub : 밑첨자

ul : 순서가 없는 목록 태그
ol : 순서가 있는 목록 태그
li : 목록 요소

TABLE
tr : 표 행태그
th : 행 내부 제목 셀 태그 (td보다 굵음) 기본 가운데 정렬
td : 행 내부 일반 셀 태그 기본 왼쪽 정렬
colspan="num" : 셀 합치기
rowspan="num" : 행 합치기

Img Tag
src : 이미지 경로 지정
alt : 이미지가 없을 때의 글 지정
width, height : 너비 높이 지정(%, px)
